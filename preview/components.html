Foundations · Components

Components

Every interactive element in Ekklo speaks the same vocabulary: pills for action and tabs, 12px cards for grouping, 8px inputs, near-black for "this is selected," lime for "this is the next step."

Buttons

One primary, one secondary, three quieter alternatives. Buttons are always pills; trailing icons (arrow, check) lean into the verb.

Primary — Lime fill

The single loudest action on a screen. One per view.

Secondary — Ink fill

Used for "Cancel" inside dialogs and the trial-banner CTA. Never paired with anything other than the primary lime button.

Outline — Lime stroke

Two-of-two action layouts: "Assign a template" + "Create a program". The outline holds the alternate path without competing with the primary fill.

Ghost & text

Quiet rails — toolbar buttons, "Cancel" outside dialogs, video tutorial links in the topbar.

Tabs & chips

Tabs are pills. The selected pill is solid ink; all others are transparent until hover. Tab groups are left-aligned under the page title.

Tab group

Used everywhere — Programs / Meal plans / Meals / Food equivalents. Single ink-filled pill marks the active view.

Pipeline / kanban tab

Same pill but with a colored dot prefix and a count. From the Prospects board.

Status badges

All chips share a 6px radius and 11px semibold type. Status color stays in the chip — never spreads.

Trial banner

Sits in the top center, hugs the page title — the trial countdown lives here so the rest of the chrome stays quiet.

Inputs

Every input is 40px tall. Search rounds to a pill; everything else uses 8px. Required fields use a small red asterisk (the only place danger-red is allowed in steady state).

Text + helper

Label above. Helper sits just under the label, never under the field.

This name will be visible on your profile. 100 characters maximum.

Search, select, toggle

Search uses the pill input — the only pill-shaped input. The selected dropdown rotates a chevron.

Sidebar navigation

240px wide, lives on the canvas (not its own surface). Three groups: top items, then RESOURCES, then MANAGEMENT, then PROFILE — each separated by an uppercase tertiary-gray eyebrow.

Anatomy

Nav items are 36px rows. Selected state uses a sunken-gray pill (not lime — the lime is reserved for action). "New" / "V2" tags sit at the right of an item.
