Foundations · Color

Color

One bright lime, one near-black, one mint for help. Everything else is grayscale. The lime is the loudest thing on any screen — use it once per view, on the next thing you want the coach to do.

How the brand pair plays out

Lime + ink is the engine of the system. The pattern is consistent across every screen: lime for "do the next thing," ink for the selected pill or the secondary action.

Pill family

Five forms, one shape. Solid lime is reserved for the screen's single primary action.

On-screen color budget

A typical Ekklo screen budgets color like this — lots of neutral space, a single lime moment, ink for navigation state, mint for the help affordance only.
