Spacing, radii & shadow

The system is built on an 8px rhythm with a 4px half-step for inline density. Pills (full radius) signal interactivity; 12px corners signal grouping; flat 0px lives only inside table rows.

Spacing scale

All padding and gap values come from this scale. --s-4 (16px) is the default card padding step; --s-6 (24px) is the default card-to-card gap.

Radii

Three radii. Anything pill-shaped is interactive; anything 12px-cornered is a content group.

Pill — 999px

Buttons, badges, tabs, search inputs, the trial banner CTA.

--r-pill

Card — 12px

Cards, modals, the sidebar getting-started box, image upload tiles.

--r-card

Input — 8px

Text inputs, selects, dropzones, code blocks. Sits between pill and card.

--r-input

Shadow

Shadows are restrained — Ekklo borders cards instead of floating them. Only modals and the help FAB get a real shadow.

Card

Almost imperceptible — pairs with the 1px border for a hairline lift.

Modal

Used on dialogs, side sheets, and the slide-over prospect panel.

FAB

Mint-tinted shadow on the help bubble — the only colored shadow in the system.

Layout grid

The app is a fixed 240px sidebar plus a fluid main column. Main content sits in a 1180px max-width container with 24px gutters.
